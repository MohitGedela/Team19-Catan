## Overview
This project develops a simulator for the game of Settlers of Catan. The simulator models the core elements of the game, including the board, players, resources, settlements, cities, and roads, while adhering to the rules specified in the course’s rulebook. The project follows object-oriented design principles, emphasizing abstraction, encapsulation, and clear class responsibilities. SOLID principles and OO Mechanisms were applied throughout.

## How to Run
- **Task 4: Simulator**  
  1. Open a terminal and navigate to the `Task4` directory.  
     ```bash
     cd Task4
     ```  
  2. Run the simulator:  
     ```bash
     java .\Catan.java
     ```

- **Task 3: Demonstrator**  
  1. Open a terminal and navigate to the Task 3 source folder:  
     ```bash
     cd Task3\src\main\java\com\boardgame
     ```  
  2. Run the demonstration program:  
     ```bash
     java .\GameDemo.java
     ```
